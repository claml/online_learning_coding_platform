# 前端白屏问题排查摘要

- **现象**：启动前后端后访问 `http://localhost:5176/` 时页面为空白。
- **原因**：Vite 默认开发端口为 5173，但当前配置未显式指定端口。前端开发服务器实际监听在 5173，而浏览器访问的是 5176，导致未命中运行中的前端服务而显示空白页面。
- **修复**：在 `frontend/vite.config.js` 中为 `server` 指定 `port: 5176`，使开发服务器与访问端口一致。
- **验证建议**：重新运行 `npm run dev`（或对应启动命令），确认日志提示开发服务器地址包含 `5176`，然后再次访问 `http://localhost:5176/`，页面应正常加载。
